# Python - Hello, World

This directory contains introductory tasks to Python programming including shebang line setups, output formatting, slicing, basic operations, and linked list cycle detection in C.

## Tasks
- **0-run**: Shell runner script.
- **1-run_inline**: Inline Python code runner.
- **2-print.py**: Prints string literal to stdout.
- **3-print_number.py**: Formatted integer printing.
- **4-print_float.py**: Formatted float printing.
- **5-print_string.py**: Slicing and repetition of strings.
- **6-concat.py**: Concatenation of strings.
- **7-edges.py**: Slicing edges of a string.
- **8-concat_edges.py**: Advanced slicing concatenation.
- **9-easter_egg.py**: Zen of Python easter egg.
- **10-check_cycle.c**: Singly linked list cycle checker in C.
- **100-write.py**: Prints to stderr.
- **101-compile**: Python file compiler shell runner.
- **102-magic_calculation.py**: Custom Python calculation matching bytecode.
