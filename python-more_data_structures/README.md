# Python - More Data Structures: Set, Dictionary

This directory contains tasks relating to sets, dictionaries, functional list manipulation (map, lambda), weighted averages, and CPython list/byte object introspection.

## Tasks
- **0-square_matrix_simple.py**: Squared values matrix.
- **1-search_replace.py**: Element substitution inside list.
- **2-uniq.py**: Adds unique elements.
- **3-common_elements.py**: Set intersection finder.
- **4-only_diff_elements.py**: Set symmetric difference finder.
- **5-number_keys.py**: Counts dictionary keys.
- **6-print_sorted_dictionary.py**: Sorted printing of key/values.
- **7-update_dictionary.py**: Adds/updates dictionary entries.
- **8-simple_delete.py**: Deletes dictionary keys.
- **9-multiply_by_2.py**: Doubles dictionary values.
- **10-best_score.py**: Gets key with highest value.
- **11-multiply_list_map.py**: List mapping elements.
- **12-roman_to_int.py**: Roman numerals parser.
- **100-weight_average.py**: Weighted average math.
- **101-square_matrix_map.py**: Matrix mapper.
- **102-complex_delete.py**: Deletes keys by value.
- **103-python.c**: CPython list and bytes inspectors.
