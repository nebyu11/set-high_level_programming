# Python - Everything is object

This directory contains tasks exploring the Python object model, variable references, and identity.

## Tasks
- **0-answer.txt** to **28-answer.txt**: Answer files analyzing Python comparisons, mutability, references, and identity.
- **19-copy_list.py**: Function returning a shallow copy of a list.
- **100-magic_string.py**: Function returning a string with BestSchool iterations.
- **101-locked_class.py**: Class restricting dynamic attributes using `__slots__`.
- **103-line1.txt** to **106-line5.txt**: Advanced variable interning identity questions.
