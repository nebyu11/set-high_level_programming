# Set High Level Programming

This repository contains Python and C programming tasks for the **SE 202: High Level Programming I** course.

## Directories

- [python-hello_world](./python-hello_world) - Introduction to Python syntax, basic output printing, compiling, and running scripts.
- [python-if_else_loops_functions](./python-if_else_loops_functions) - Branching, loops, functions, and linked list insertions in C.
- [python-import_modules](./python-import_modules) - Importing functions, variables, modules, command-line arguments parsing, and CPython internals.
- [python-data_structures](./python-data_structures) - Lists, tuples, matrix processing, linked list palindrome checking in C, and CPython list information.
- [python-more_data_structures](./python-more_data_structures) - Dictionaries, sets, list mapping, weighted averages, and CPython list and byte object introspection.
