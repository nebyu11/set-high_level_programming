# Python - More Classes and Objects

This directory contains advanced tasks focusing on classes and objects in Python.

## Tasks
- **0-rectangle.py**: Empty `Rectangle` class defining a rectangle.
- **1-rectangle.py**: `Rectangle` class with private `width` and `height` properties.
- **2-rectangle.py**: `Rectangle` class adding `area()` and `perimeter()` methods.
- **3-rectangle.py**: `Rectangle` class with string representation printing `#`.
- **4-rectangle.py**: `Rectangle` class adding representation methods (`__repr__`).
- **5-rectangle.py**: `Rectangle` class printing deletion message on garbage collection.
- **6-rectangle.py**: `Rectangle` class tracking active instance counts.
- **7-rectangle.py**: `Rectangle` class implementing customizable print symbol configurations.
- **8-rectangle.py**: `Rectangle` class adding static area comparison methods.
- **9-rectangle.py**: `Rectangle` class adding class methods to instantiate squares.
- **101-nqueens.py**: Chess backtracking solver calculating all solutions for N-Queens.
