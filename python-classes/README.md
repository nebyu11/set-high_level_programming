# Python - Classes and Objects

This directory contains tasks relating to object-oriented programming (OOP) in Python.

## Tasks
- **0-square.py**: Empty `Square` class defining a square.
- **1-square.py**: `Square` class with private instance attribute `size`.
- **2-square.py**: `Square` class validating `size` type and values.
- **3-square.py**: `Square` class with `area()` method returning current square area.
- **4-square.py**: `Square` class utilizing property getter and setter for `size`.
- **5-square.py**: `Square` class with `my_print()` to print the square using `#`.
- **6-square.py**: `Square` class adding positional indentation to printing.
- **100-singly_linked_list.py**: Singly-linked list structure using `Node` and `SinglyLinkedList`.
- **101-square.py**: `Square` class implementing string formatting representation.
- **102-square.py**: `Square` class defining standard numeric area comparison operators.
- **103-magic_class.py**: Reverse-engineered circle representation matching bytecode.
