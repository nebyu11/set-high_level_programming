# Python - Import & Modules

This directory focuses on modular programming, importing functions, handling CLI arguments, inspecting attributes using dir(), and building calculators.

## Tasks
- **0-add.py**: Import and addition.
- **1-calculation.py**: Basic calculations using imported functions.
- **2-args.py**: Parsing and printing CLI arguments.
- **3-infinite_add.py**: Infinite addition of CLI arguments.
- **4-hidden_discovery.py**: Hidden names discovery in a pyc module.
- **5-variable_load.py**: Variable import and print.
- **100-my_calculator.py**: Interactive command-line calculator.
- **101-easy_print.py**: Print bypass using low-level os writes.
- **102-magic_calculation.py**: Custom logic matching bytecode imports.
- **103-fast_alphabet.py**: Fast printing of uppercase alphabet.
