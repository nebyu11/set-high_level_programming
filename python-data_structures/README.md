# Python - Data Structures: Lists, Tuples

This directory contains operations on lists and tuples, matrix element formatting, a palindrome checker for C linked lists, and CPython list info.

## Tasks
- **0-print_list_integer.py**: Print integers of a list.
- **1-element_at.py**: Retrieve list elements in C-style bounds.
- **2-replace_in_list.py**: In-place replacement of list elements.
- **3-print_reversed_list_integer.py**: Print reversed list integers.
- **4-new_in_list.py**: Replace element in copy of a list.
- **5-no_c.py**: Removes 'c' and 'C' from string.
- **6-print_matrix_integer.py**: Matrix of integers printer.
- **7-add_tuple.py**: Addition of two tuples.
- **8-multiple_returns.py**: Returns size and first letter.
- **9-max_integer.py**: Find maximum integer in list.
- **10-divisible_by_2.py**: Finds multiples of 2.
- **11-delete_at.py**: Deletes item at index.
- **12-switch.py**: Swaps variable values.
- **13-is_palindrome.c**: C list palindrome checker.
- **100-print_python_list_info.c**: CPython list details printer.
