# Python - Inheritance

This directory contains tasks exploring object-oriented inheritance, superclasses, subclasses, and built-in class functions.

## Tasks
- **0-lookup.py**: Function returning the list of available attributes and methods of an object.
- **1-my_list.py**: List subclass implementing a public sorted printing method.
- **2-is_same_class.py**: Checks if an object is exactly an instance of a class.
- **3-is_kind_of_class.py**: Checks if an object is an instance or subclass instance.
- **4-inherits_from.py**: Checks if an object is an inherited subclass instance.
- **5-base_geometry.py**: Empty BaseGeometry class.
- **6-base_geometry.py**: BaseGeometry class with unimplemented `area()`.
- **7-base_geometry.py**: BaseGeometry class adding value validator logic.
- **8-rectangle.py**: Rectangle subclass inheriting from BaseGeometry.
- **9-rectangle.py**: Full Rectangle implementation adding custom representation.
- **10-square.py**: Square subclass inheriting from Rectangle.
- **11-square.py**: Full Square implementation with custom representation.
- **100-my_int.py**: Rebel subclass inverting equality and inequality check operators.
- **101-add_attribute.py**: Function adding a new attribute if possible.
