# Python - If/Else, Loops, Functions

This directory covers conditional structures (if/else), loops (for, while), functions, C singly-linked list node insertion, and alternating case printing.

## Tasks
- **0-positive_or_negative.py**: Sign check of random numbers.
- **1-last_digit.py**: Python correct last-digit calculations.
- **2-print_alphabet.py**: Prints lowercase alphabet.
- **3-print_alphabt.py**: Prints alphabet excluding specific characters.
- **4-print_hexa.py**: Prints numbers in hex format.
- **5-print_comb2.py**: Prints 2-digit combinations.
- **6-print_comb3.py**: Prints combinations of two digits.
- **7-islower.py**: Checks if character is lowercase.
- **8-uppercase.py**: Prints string in uppercase.
- **9-print_last_digit.py**: Prints and returns last digit.
- **10-add.py**: Adds two integers.
- **11-pow.py**: Calculates base to the power exponent.
- **12-fizzbuzz.py**: FizzBuzz printer.
- **13-insert_number.c**: Inserts number in sorted list in C.
- **100-print_tebahpla.py**: Reverse alternating case alphabet.
- **101-remove_char_at.py**: Removes character at index.
- **102-magic_calculation.py**: Math operations matching bytecode.
